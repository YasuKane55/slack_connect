# 🏙 プーブタウン 設定資料 ―― お店と人

> ゲーム『ぷーぶ君の まちたんけん』の舞台「プーブタウン」に登場する
> お店・人(NPC)の設定集。**公式キャラはぷーぶ君のみ**／他はすべて
> オリジナル・簡素な見た目でOK（仮ビルボード→必要に応じ調整）。
> 各キャラはこどもの“好き”の柱(ものづくり/農業/お店/デザイン/しぜん…)の
> やさしいロールモデルになるよう設計。たたき台なので自由に追加・変更可。

最終更新: 2026-06-11 / v1

## 🗺 街のエリア構成

```
            [ じゅうたく ]                [ はたけ・農園 ]
        ぷーぶ君の家・友達の家            ぐんぐんファーム
                 │                            │
   [ こうえん ]──┼──[ ちゅうおう広場 ]──┼──[ おみせ通り ]
   池・はまべ      (ふんすい・掲示板)        いろんなお店がならぶ
                 │                            │
        [ がくしゅう ]                 [ ものづくり ]
      としょかん・スキルがっこう          アトリエ・こうぼう
```

- **ちゅうおう広場**：街の中心。ふんすい＆掲示板（デイリークエスト・称号確認）
- **おみせ通り**：お店がならぶにぎやかな通り
- **はたけ・農園**：野菜や花を育てる（農業の柱）
- **ものづくりエリア**：アトリエ（ものづくり・デザインの柱）
- **がくしゅうエリア**：としょかん・スキルがっこう
- **こうえん／池／はまべ**：採集・つり（しぜん観察の柱）
- **じゅうたく**：ぷーぶ君の家、友達の家（部屋づくり・訪問）

## 🧑‍🤝‍🧑 登場キャラクター（詳細設定）

> 見た目はすべて「仮・簡素なビルボード」想定。種族や色はイメージ用の案。

### ⭐ ぷーぶ君（主役・プレイヤー）
- **役わり**：プレイヤーが操作する主人公
- **見た目**：公式イラスト（PNG）を使用
- 街のみんなと関わりながら、自分の“好き”を見つけていく

---

### 1. モグさん（よろずや店主）
- **種族/見た目**：もぐら。茶色のまるい体に大きなしっぽエプロン
- **柱**：お店（商い）
- **性格**：せかせか働き者。世話やき
- **あいさつ**：朝「らっしゃい！朝はパンが合うよ」／夜「もう店じまいだよ〜」
- **すきなもの**：とれたて野菜（ファームの収穫物）
- **いる時間/場所**：朝〜夕はモグモグ商店、夜は広場で一服
- **クエスト例**：
  - ★1「品物をパンやさんに届けて」（配達）
  - ★2「えさを10こ仕入れたいから貝をひろってきて」（収集）

### 2. パン蔵（パンやのコック）
- **種族/見た目**：くま。白いコック帽、ふっくら
- **柱**：お店／料理
- **性格**：おおらかで職人気質。焼きたてに命をかける
- **あいさつ**：朝「いいにおいするだろ？焼きたてだよ」
- **すきなもの**：りんご・とれたて小麦
- **いる時間/場所**：朝・昼はベーカリー
- **クエスト例**：
  - ★2「アップルパイに りんごが2こ いるんだ」（収集）
  - ★3「朝市に出すパンを いっしょに焼こう」（おてつだい・ミニゲーム）

### 3. ハナさん（おはなやさん）
- **種族/見た目**：うさぎ。花かんむり、エプロン
- **柱**：農業／デザイン（花でかざる）
- **性格**：おっとり、やさしい。花が大好き
- **あいさつ**：昼「きょうの おすすめは チューリップだよ」
- **すきなもの**：きれいな花・めずらしいたね
- **いる時間/場所**：昼・夕はフラワー店、朝はファームで水やり
- **クエスト例**：
  - ★1「おはなを1つ プレゼントして」（プレゼント）
  - ★2「広場をお花でかざりたいから5本あつめて」（収集・デザイン）

### 4. リリィ（おもちゃやさん）
- **種族/見た目**：ねこ。リボン、カラフルな服
- **柱**：ものづくり／デザイン
- **性格**：あかるくて元気。アイデアいっぱい
- **あいさつ**：昼「あたらしいおもちゃ 入ったよ！」
- **すきなもの**：手作りの作品（アトリエで作ったもの）
- **いる時間/場所**：昼・夕はトイズ
- **クエスト例**：
  - ★2「こわれたおもちゃを直す材料をあつめて」（収集）
  - ★3「子どもたちに人気のおもちゃをデザインして」（つくる）

### 5. ミルクさん（カフェ店主）
- **種族/見た目**：ひつじ。もこもこ、やさしい色のエプロン
- **柱**：料理／人と関わる
- **性格**：おだやかで聞き上手。みんなの休憩所
- **あいさつ**：夕「つかれたら 寄っていってね」
- **すきなもの**：とれたてミルク・くだもの
- **いる時間/場所**：昼・夕・夜はカフェ
- **役わり**：飲みもの/スイーツで **きぶん回復**。街のうわさ・ヒントが聞ける
- **クエスト例**：
  - ★1「常連さんに ことづけを とどけて」（おつかい）
  - ★3「みんなの好きな味を聞いて 新メニューを考えよう」（おてつだい・なかなおり）

### 6. モードさん（ブティック店主）
- **種族/見た目**：くじゃく。はなやかな羽、おしゃれな帽子
- **柱**：デザイン
- **性格**：自由でセンスばつぐん。ほめ上手
- **あいさつ**：昼「きょうの きみ、にあってるよ！」
- **すきなもの**：めずらしい色・きれいな素材
- **いる時間/場所**：昼・夕はブティック
- **役わり**：服の着せ替え、**部屋やお店・看板のデザイン相談**（デザインの柱の中心）
- **クエスト例**：
  - ★2「お店の看板をいっしょにデザインして」（つくる・デザイン）
  - ★4「街の かざりつけを プロデュースして」（連鎖・デザイン）

### 7. ホー先生（としょかんのフクロウ）
- **種族/見た目**：フクロウ。まるメガネ、本をかかえる
- **柱**：しぜん観察／学び
- **性格**：もの知りでおだやか。なぞなぞ好き
- **あいさつ**：夜「夜は本を読むのに いい時間じゃ」
- **すきなもの**：めずらしい本・図鑑のうめ合わせ報告
- **いる時間/場所**：ほぼ一日としょかん（夜もあいてる）
- **役わり**：ヒント・図鑑の管理・むし/さかなの解説
- **クエスト例**：
  - ★1「なくした本を さがして」（さがしもの）
  - ★3「むしを5種つかまえて図鑑をうめて」（収集・しぜん観察）

### 8. マナブ先生（スキルがっこう）
- **種族/見た目**：やぎ。めがね、ちょっと立派なヒゲ
- **柱**：（全部の入口）スキル習得の案内人
- **性格**：まじめで熱心。生徒思い
- **あいさつ**：朝「きょうは なにを 学ぶかな？」
- **いる時間/場所**：朝・昼はがっこう
- **役わり**：プーブを払ってスキル習得（おかいもの名人/たんけん家/つり名人/ものづくり 等）
- **クエスト例**：
  - ★2「スキルの れんしゅうを 見せて」（おてつだい）

### 9. アートさん（アトリエのアーティスト）
- **種族/見た目**：きつね。絵の具のついたエプロン、ベレー帽
- **柱**：ものづくり（中心人物）
- **性格**：マイペースで情熱的。作るのが大好き
- **あいさつ**：昼「きょうは何を つくろうか？」
- **すきなもの**：きれいな素材・できあがった作品
- **いる時間/場所**：昼・夕はアトリエ
- **役わり**：**ワークショップ開催**（材料＋プーブで作品づくり→部屋に飾れる）
- **クエスト例**：
  - ★2「お母さんへの絵を いっしょに作ろう」（つくる）
  - ★4「街のお祭りの かざりを みんなで作ろう」（連鎖・大イベント）

### 10. ターボさん（ぐんぐんファーム）
- **種族/見た目**：りす。麦わら帽子、長ぐつ（名前は“すばやい”の意味）
- **柱**：農業（中心人物）
- **性格**：元気いっぱい、はやくち。土いじりが大好き
- **あいさつ**：朝「朝は水やりからだ！手伝ってくれ〜」
- **すきなもの**：めずらしいたね・大きく育った野菜
- **いる時間/場所**：朝・昼はファーム
- **役わり**：**たね・農具を売り、育て方を教える**（種まき→水やり→収穫）
- **クエスト例**：
  - ★1「トマトを 3こ 収穫して」（収集・農業）
  - ★3「畑のやさいで 料理を作って みんなにふるまおう」（連鎖）

### 11. うみのおじさん（池・はまべの釣り名人）
- **種族/見た目**：かわうそ。むぎわら帽、つりざお
- **柱**：しぜん観察（釣り）
- **性格**：のんびり、こわくない頼れる大人
- **あいさつ**：夕「夕方は 大物が つれるぞ」
- **すきなもの**：大きな魚・めずらしい貝
- **いる時間/場所**：朝・夕・夜は水辺
- **役わり**：つりの手ほどき・つり道具・さかな図鑑
- **クエスト例**：
  - ★2「魚を 3びき つって」（収集）
  - ★4「まぼろしの大物を つりあげろ」（高難度・夜限定）

### 12. ミミ（こどもの友達）
- **種族/見た目**：うさぎの女の子。おさげ
- **柱**：人と関わる（友達）
- **性格**：やさしくて しっかり者
- **あいさつ**：昼「いっしょに あそぼ！」
- **すきなもの**：おはな・あまいもの
- **いる時間/場所**：昼は広場・公園、夕は家
- **クエスト例**：
  - ★1「かくれんぼ しよう」（おてつだい・ミニゲーム）
  - ★3「ケンカした タロと なかなおりさせて」（なかなおり）

### 13. タロ（こどもの友達）
- **種族/見た目**：いぬの男の子。ぼうし、元気
- **柱**：しぜん観察（むしとり大好き）
- **性格**：やんちゃで好奇心おうせい
- **あいさつ**：朝「むし、いっぱい とるぞー！」
- **すきなもの**：めずらしいむし・おもちゃ
- **いる時間/場所**：朝・昼は公園
- **クエスト例**：
  - ★2「いっしょに カブトムシを さがそう」（収集・しぜん観察）

### 14. おまわりのポリィ（街の見回り）
- **種族/見た目**：犬のおまわりさん。青いぼうし
- **柱**：人と関わる
- **性格**：まじめで親切。街の安全を守る
- **あいさつ**：いつでも「街で こまったこと ないかい？」
- **いる時間/場所**：時間帯で街を見回り（場所が変わる）
- **クエスト例**：
  - ★2「まいごの 子ねこを さがして」（さがしもの）
  - ★3「落とし物を 持ち主に とどけて」（おつかい）

### 15. ポポ町長（プーブタウンのリーダー）
- **種族/見た目**：恰幅のいい動物（案：たぬき）。立派なリボン
- **柱**：街全体（メインの進行役）
- **性格**：あたたかく、街とこどもが大好き
- **あいさつ**：「プーブタウンへ ようこそ！」
- **いる時間/場所**：ちゅうおう広場（掲示板のそば）
- **役わり**：**称号の授与・街の発展イベント・大きな目標の提示**
- **クエスト例**：
  - ★1（最初）「街を ひとまわり たんけんしてみよう」（チュートリアル）
  - ★5「みんなで 街のお祭りを 成功させよう」（大イベント・協力）
