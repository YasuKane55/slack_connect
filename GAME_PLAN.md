# 🎮 ゲーム企画・設計書『ぷーぶ君の まちたんけん』

> こどもプーブリカ オリジナルキャラクター「ぷーぶ君」が主役の、
> 独自通貨「プーブ」で遊ぶ子ども向け生活アドベンチャーゲーム。

最終更新: 2026-06-11 / バージョン: プランv1（プロトタイプ前）

---

## 1. コンセプト

ぷーぶ君になって「**プーブタウン**」を探検し、独自通貨 **プーブ** を
稼いだり使ったりしながら、まちの人と関わって少しずつできることを
増やしていく、ほのぼの生活ゲーム。

- 対象：**小学生中心**（ひらがな多め＋漢字にふりがな相当のやさしさ）
- 学びの要素：お金（プーブ）のやりくり、人の役に立つよろこび
- 雰囲気：あつ森のような、かわいくて立体感のある世界

## 2. コアループ（遊びの基本の流れ）

```
        まちを探検する
            │
            ▼
   まちの人の困りごとを見つける ──► 解決する（クエスト）
            │                          │
            ▼                          ▼
        お買い物 ◄──── プーブを稼ぐ・使う ────► 自分のお店で稼ぐ
            │                          │
            ▼                          ▼
        スキルを習得（できること解放）        │
            │                          │
            └────────► 称号がもらえる ◄───────┘
```

## 3. 確定した方針（技術・表現）

| 項目 | 決定 | 補足 |
|---|---|---|
| プラットフォーム | Web ブラウザ | PC/スマホ/タブレット。URL共有で友達に渡せる |
| 表現 | 見下ろし型 **本格3D** | あつ森っぽい立体感をめざす |
| 3Dエンジン | **Three.js**（CDN読み込み） | ビルド不要。`importmap` で読み込み |
| アートスタイル | 低ポリ＋トゥーン調 | リアルより「かわいい立体」 |
| カメラ | 斜め見下ろし（追従） | やわらかい影と光 |
| キャラ表現 | **ビルボード方式** | 公式PNGイラストを常にカメラへ向けて立てる |
| 素材 | **PNG画像**をリポジトリ配置 | `assets/` に置く |
| セーブ | localStorage（端末内） | のちにサーバー保存へ拡張 |
| 公開 | GitHub Pages 等 | 静的サイトとしてそのまま置ける |
| オンライン化 | 将来：Node.js + WebSocket | 同じまちに集合＆チャット |

> ⚠️ 期待値メモ：あつ森は大規模スタジオが数年かけた3Dゲームです。
> 同等品質ではなく「あつ森**っぽい**かわいい立体世界を歩く体験」を狙います。

## 4. 3Dシーン設計

- **地面**：タイル状（草・道・広場・水辺）。低ポリのメッシュ＋やわらかい色
- **建物**：箱＋屋根の低ポリ。入口に立つとパネルが開く
- **装飾**：木・花・ベンチ・街灯・池など（インスタンス描画で軽く）
- **ライティング**：環境光＋やわらかい平行光（影あり）。空は淡いグラデ
- **カメラ**：プレイヤーを追う斜め見下ろし。ホイール/ピンチでズーム
- **キャラ**：PNGをビルボードのスプライトに貼り、進行方向で左右反転

## 5. ゲーム内容（コンテンツ設計）

### 5-1. まちの場所（マップ）
- 🏠 ぷーぶ君のおうち（スタート地点・セーブ）
- 🛒 よろずや（食べ物・日用品を買う）
- 🧸 おもちゃやさん
- 🌸 おはなやさん
- 📚 としょかん（本）
- 🏫 スキルがっこう（スキルを習う）
- 🏪 じぶんのおみせ（スキル習得後に開放）
- 🌳 こうえん・池（探検＆イベント）
- 🟡 ふんすい広場（まちの中心・掲示板）

### 5-2. アイテム例
| ID | なまえ | ねだん(プーブ) | 用途 |
|---|---|---|---|
| ringo | りんご | 5 | クエスト/食べ物 |
| pan | パン | 8 | 食べ物 |
| hana | おはな | 6 | クエスト/プレゼント |
| hon | ほん | 12 | クエスト |
| kasa | かさ | 15 | クエスト |
| omocha | おもちゃのくるま | 20 | プレゼント |
| present | プレゼント | 25 | クエスト |

### 5-3. クエスト（まちの人の困りごと）例
| 依頼人 | こまりごと | 条件 | ごほうび |
|---|---|---|---|
| ハナさん | おはなが ほしい | おはな×1 を わたす | 15プーブ |
| パンやのコック | パイに りんごが いる | りんご×2 を わたす | 20プーブ＋割引のヒント |
| こどものミミ | ほんが よみたい | ほん×1 を わたす | 10プーブ |
| おじいさん | かさを なくした | かさ×1 を わたす | 18プーブ |

### 5-4. スキル（スキルがっこうで習得・プーブ払い）
| スキル | こうか | ねだん | 条件 |
|---|---|---|---|
| おかいもの名人 | お店が10%やすくなる | 30 | - |
| おしゃべり上手 | かくれたクエストが出る | 20 | - |
| おみせやさん | じぶんのおみせを開ける | 40 | - |
| たんけん家 | はやく歩ける/池の小道が開く | 25 | - |

### 5-5. 称号（条件達成で自動付与）
| 称号 | 条件 |
|---|---|
| はじめてのおかいもの | 1回お買い物した |
| プーブもち | プーブを100以上もった |
| まちのヒーロー | クエストを3つ解決した |
| おみせやさん | 自分のお店を開いた |
| たんけんマスター | まちの全エリアを訪れた |
| プーブタウンの主役 | すべての称号を集めた |

### 5-6. 自分のお店
- スキル「おみせやさん」習得後に開放
- もちものから売る品物を「ならべる」→「おきゃくさんを よぶ」でプーブ獲得
- 仕入れ値より高く売れる＝お金のやりくりを学べる
