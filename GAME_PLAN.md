## 0. このゲームのねらい（目的・ビジョン）★最重要

> **こどもたちがゲームを通して、「街と関わることの楽しさ」を知り、
> 自分の“好きなこと”（ものづくり・農業・お店・デザイン など）を
> 見つけるきっかけになること。**

このゲームは「クリアして終わり」ではなく、**いろいろな“好き”を安心して
試せる場所**です。設計のすべての判断は、次の3つのねらいに立ち返ります。

1. **街と関わる楽しさ** … 人と関わり、誰かの役に立ち、街が育つ手ごたえ
2. **“好き”との出会い** … ものづくり/農業/お店/デザイン…を対等に試せる
3. **自分のペースで** … 競争や正解より、自分の興味で進める安心感

### 設計の原則（このねらいを守るためのルール）
- ❌ 失敗で罰しない／急かさない（こどもが安心して試せる）
- ✅ 「どれを選んでもいい」…活動に優劣をつけない（§1-2の柱は対等）
- ✅ 関わるほど世界が広がる（街・人・できることが増える）
- ✅ 押し付けず、そっと気づける（§1-3 自己発見の仕組み）
- ✅ 作ったもの・育てたものが**残って飾れる**＝自分の足あとになる

## 1. コンセプト

ぷーぶ君になって「**プーブタウン**」で暮らし、独自通貨 **プーブ** を
稼いだり使ったりしながら、まちの人と関わり、ごはんを食べたり眠ったり、
ものを作って部屋を飾ったりして、少しずつできることを増やしていく
**ほのぼの生活アドベンチャー**。

- 対象：**小学生中心**（ひらがな多め・やさしい言葉）
- ねらい（§0）：街と関わる楽しさ／自分の“好き”を見つけるきっかけ
- 育つもの：お金のやりくり、生活リズム、人の役に立つよろこび、ものづくり・農業・商い・デザインへの興味
- 雰囲気：あつ森のような、かわいくて立体感のある「住める世界」

## 1-2. “好き”を見つける「かつどうの柱」

こどもが自分の興味で選べる、**対等な活動の柱**。どれを選んでもOK、
かけ持ちもOK。それぞれに浅い入口〜深いやり込みを用意します。

| 柱 | ゲーム内の体験 | つながる“好き”の芽 |
|---|---|---|
| 🔨 **ものづくり** | アトリエのワークショップ、工作・作品づくり | クラフト・図工・エンジニア |
| 🌱 **農業（はたけ）** | たねをまき、水やり・収穫、育てて売る/料理 | 自然・栽培・食 |
| 🛒 **お店（商い）** | 仕入れ・値づけ・陳列して稼ぐ自分の店 | 商売・経営・接客 |
| 🎨 **デザイン** | 部屋・お店・看板・飾りのコーディネート | 美術・空間/グラフィックデザイン |
| 🍳 **料理**（任意拡張） | 収穫物で料理、ふるまう | 食・おもてなし |
| 🎣 **しぜん観察**（採集/釣り/図鑑） | むし・さかな・植物を集めて記録 | 生物・研究・収集 |

> 関わるほど深まる。こどもが自然と通いたくなる柱を見つけられる。

## 6-2. 街を「生きた場所」にする（最初の作り込みの中心）

ねらい「街と関わる楽しさ」のため、お店と人を**最優先で充実**させる。
各NPC／お店に、最低これだけ持たせると街がぐっと生きてくる👇

- **あいさつ会話**：時間帯や天気で少し変わる一言（朝「おはよう！」夜「もう夜だね」）
- **役わり**：お店なら売り買い、職人ならワークショップ、子どもなら遊び など
- **クエスト**：1人につき1つ以上（難易度はバラけさせる／称号で増える §15）
- **いる時間・場所**：時間帯で居場所が変わる簡単スケジュール（昼はお店、夕方は広場 等）
- **すきなもの**：プレゼントや好物に反応（きぶんUP・仲良し度UP）
- **見た目**：オリジナル・簡素でOK（仮ビルボード→必要に応じ調整）

> まずは「あいさつ＋お店 or クエスト」だけでも全員に用意し、
> 街を歩くと誰かしらと関われる“にぎわい”を最初に作る。

## 8. ぷーぶ君のコンディション（生活シム要素）

3つのゲージを **やさしく** 管理します ✅決定（こども向けなので「死ぬ・ゲームオーバー」は無し）。

| ゲージ | 下がると… | 回復方法 |
|---|---|---|
| 🍙 **おなか**（満腹度） | 0に近いと歩くのが おそく / はしれない | 食べ物を たべる |
| 😴 **ねむけ**（げんき） | 低いと きぶんが下がりやすい / はしれない | おうちで ねる＝ねむけ回復（時間はスキップしない）。ベンチで少し |
| 😊 **きぶん**（きもち） | 低いと まちの人の反応がしょんぼり | 好物を食べる・クエスト達成・部屋を飾る・友だちと遊ぶ |

- 時間とともに おなか・ねむけ が少しずつ減る（ゆっくり）
- **ペナルティは軽め**：困っても「ちょっと しんどい」程度。寝れば/食べればすぐ回復
- 上級者向けに「がんばりボーナス」：3つとも高いと採集や釣りの成果UP（任意）

## 9. 時間と一日の流れ ✅決定

- **統一されたゲーム内標準時間**を持つ（友達と同期して遊ぶ前提）
  - オンライン時は **サーバーが時刻の親**。全員が同じ時間・同じ空の下で遊ぶ
  - プロトタイプ(1人用)では端末内の同じルールで時計を回し、後でサーバー時刻に差し替え
- **実世界とは連動しない**。**リアルより早く進む**
- **時間帯が30分ごとに切り替わる** ✅決定（朝→昼→夕方→夜→朝…、各30分）。1日一周＝約2時間
- **朝・昼・夕・夜** で空の色・BGM・お店の開閉が変化
- ひと遊び（30分〜）で時間帯が移ろう、ゆったりめのテンポ
- 一部クエストや採集物は時間帯限定（朝だけのむし、夜だけの魚 など）
- ⚠️ 時間は全員共有のため、**「寝て翌朝までスキップ」はしない**（時刻を飛ばせない）。
  「ねる」は §8 のとおり **ねむけ回復** の手段（時間は進めない）

## 10. クエストの種類（困りごと解決 ＝「クエスト」）

| 種類 | 内容 | 例 |
|---|---|---|
| ① おつかい/配達 | アイテムを買って/運んで渡す | よろずやの品をパンやへ届ける |
| ② 収集 | 指定アイテムを集めて渡す | りんご×2、おはな×1 |
| ③ さがしもの | なくし物・まいごを見つける | としょかんの本、こうえんの子ねこ |
| ④ おてつだい | 作業を手伝う（ミニゲーム） | 花の水やり、店の片づけ |
| ⑤ つくる | ワークショップで作って届ける | 「お母さんへの絵」を作って渡す |
| ⑥ なかなおり | 人と人をつなぐ会話チェーン | けんかした友だちを仲直りさせる |
| ⑦ デイリー/イベント | 掲示板の毎日依頼・お祭り等 | 「きょうの おつかい」、夏祭り |

- クエストは **掲示板 or NPC会話** で受注 →「クエスト帳」で進行確認
- 報酬：プーブ／アイテム／スキルのヒント／称号の進行／部屋の飾り

### 難易度と報酬（むずかしいほど たくさんプーブ）
| 難易度 | 内容のめやす | 報酬プーブ | 解放のめやす |
|---|---|---|---|
| ★1 | すぐ終わるおつかい | 5〜15 | 最初から |
| ★2 | 少し集める/さがす | 15〜30 | 称号1つ〜 |
| ★3 | 複数手順・ミニゲーム | 30〜60 | 称号3つ〜 |
| ★4 | つくる/なかなおり等の連鎖 | 60〜120 | 称号5つ＆スキル必要 |
| ★5 | まちの大イベント | 120〜＋レア飾り | 称号多数＋特定スキル |

### 称号でクエストが「育つ」（成長システム）
- 称号を取るほど、NPCが出す **クエストのバリエーションと難易度が解放**される
  - 例：「まちのヒーロー」獲得 → ★3クエストや新NPCの依頼が出はじめる
  - 例：「つりはかせ」獲得 → 釣り系の高難度クエストが解放
- 同じNPCでも、信頼が上がると（クエスト達成数で）依頼が高度化・高報酬化
- スキルが前提のクエストもある（例：おみせやさん習得→お店経営クエスト）
- これにより「やることが尽きず、進めるほど世界が広がる」

## 10-2. クエスト発注の「条件」＆ ヒントシステム（RPG要素）

### 街をふらふら歩くNPC（街の人）
- お店の店主とは別に、**街を自由に歩く“街の人”NPC**がいる（§TOWN「街の人」）
- 話しかけると **ヒント** をくれたり、**条件を満たすと クエストを発注** してくれる
- 居場所が決まっていない＝歩いて出会う楽しみ（村人感・世界が生きてる感）

### クエスト発注の条件（条件を満たすと出る）
NPC/お店のクエストには **発注条件** を持たせ、満たすと会話で受注できる：
- 🏅 **称号**：特定の称号を取得後に解放（§15 称号で育つ）
- ⭐ **スキル**：必要スキル習得後（例：たんけん家→近道クエスト）
- 💛 **仲良し度**：そのNPCと仲良くなると 高度・特別なクエスト
- ✅ **進行**：別クエストのクリア後（連鎖）
- 🕐 **時間/曜日**：朝だけ・夜だけ・特定の日（朝市など）
- 🎒 **所持/図鑑**：特定アイテム所持・図鑑の進捗（むし○種など）
> これにより「進めるほど 新しい依頼が出てくる」RPGらしい広がりに。

### ヒントシステム（村人に聞ける）
- 街の人や案内役に話しかけると、**今の状況に応じたヒント**がもらえる
  - 次に何をすべきか／クエストの進め方／かくれた場所
  - 図鑑の埋め方（アート/ねこ/マンホール/むし/さかなの在りか）
- ヒントは **状況で変化**（アクティブなクエスト・未発見の収集物に反応）
- 強い案内役：こどもプーブリカ(進行)／さんぽや(街歩き)／ネンシャ(かくれ名所)／
  本のすみか(知識)＋ふらふらNPC各人（得意分野のヒント）
- ねらい(§0)に沿って **「教えすぎず、そっと背中を押す」**ヒントに

## 11. ワークショップ（アーティストと作る）

- アトリエで **アートさん** が定期的にワークショップを開催
- **材料を集める＋プーブを払う** → 一緒に作品を完成（絵・置物・工作など）
- 完成品は **じぶんの部屋に飾れる**（コレクション／称号対象）
- 作る過程に かんたんなミニゲーム（色をぬる・組み立てる 等）を入れて達成感
- クエスト⑤「つくる」とも連動（作った物を誰かに贈る）

## 16-2. 友達と遊ぶ（オンラインのたのしみ）✅方針決定

将来のオンライン化で実現したい、友達との関わり方（プーブは実世界連動なし）。

| やりたいこと | 内容 |
|---|---|
| 🏠 **部屋を見せ合う** | 友達の部屋に遊びに行き、飾った家具/作った作品を見る。「いいね」的リアクション |
| 🏪 **お店で売り買い** | 自分のお店の商品を友達に売る／友達のお店で買う（プーブで取引） |
| 🤝 **共同クエスト** | 友達と一緒に達成する協力クエスト（みんなで収集・おてつだい・大イベント） |

> ねらい(§0)に沿って「街を一緒に育てる・見せ合う・助け合う」体験に。
> 競争より協力。プーブのやり取りは安全に（チート/不正対策はサーバー側で）。

## 17-2. 長く遊べる仕組み（やり込み・プログレッション）

「長い時間あそべる」を支える、終わりのない動機づけ 👇

- **称号ツリー**：称号→新クエスト/新エリア解放（§15）。進めるほど世界が広がる
- **NPC信頼度**：同じ人と関わるほど依頼が高度化・高報酬化、新しい会話が増える
- **デイリー/ウィークリー**：掲示板に毎日・毎週の新しい依頼（飽きずに通える）
- **きせつ・イベント**：時間/日付に応じたお祭りや限定クエスト・限定飾り
- **図鑑コンプ**：魚・むし・作品を集めきる長期目標
- **まちの成長**：プレイヤーの活躍でまちが少しずつ発展（新しいお店が開く等）
- **おみせ経営**：品ぞろえ・値づけを工夫して稼ぎを伸ばす（数値的な成長）
- **部屋づくり**：飾りを集めて理想の部屋へ（自己表現の長期目標）

> こども向けなので「急かさない」設計：自分のペースで少しずつ。
> 毎回ログインするとちょっと進む＝続けたくなる、を狙う。

## 17-2b. 街の発展度システム（まちが育つ）

> プレイヤーの貢献で **街が少しずつ発展**し、**空き物件が減って みんなが喜ぶ**。
> 「自分の行動で街がよくなる」手応え＝ねらい§0「街と関わる楽しさ」の核。

### しくみ
- **発展度（まちレベル / 発展ゲージ）** を持つ。次の貢献で **発展ポイント**が貯まる：
  - 🧹 清掃・ペイント（アンドソー系）／🎨 アート設置／🌱 空き地の緑化・花
  - 🏚 **空き物件の再生**（空き家/工場/空き地→お店/住居/広場に）
  - 🏗 建設（いえなり）／🎪 イベント成功（朝市・フェスタ等）／🛒 お店の繁盛
  - 📜 クエスト達成・困りごと解決（街への貢献全般）
- 発展度が一定に達すると **マイルストーン解放**：
  - 🏠 **空き家・空き工場・空き地が 順に 再生** → **空き物件が減る**
  - 🆕 新しいお店・住人・公園・施設が増える／街路樹・飾りで景観が華やぐ
  - 😊 住人のセリフが明るく変化・街全体のきぶんUP・**みんなが喜ぶ**
  - 🔓 新エリア/新クエスト/新図鑑の解放

### 見せ方
- **こどもプーブリカの掲示板**に「まちの発展度」メーター。節目で **ミッキー／おき町長役** が祝う
- 「前は空き家だったあそこ、お店になったね！」と **ビフォーアフター**が分かるように
- ※ **発展度は各自の街ごと（個人スコープ）**。オンラインでも合算・共有しない。
  友達の発展した街を **訪問して見る**ことで刺激し合う（詳細は §17-2c）

### データ（イメージ）
```js
townDev = {
  level: 1, points: 0, nextThreshold: 100,
  contributions: { clean, art, green, renovate, build, event, shop, quest },
  milestones: [ { atLevel, unlock:"renovate:vacant_house_1" }, ... ],
}
vacantProperty.state = "vacant" -> "renovating" -> "done(new use)"  // 発展度に連動
```

## 17-2c. オンラインの世界モデル（街は各自・訪問でつながる）★重要

> 発展度や部屋づくりは **人によって違う**。そこで オンラインは「ひとつの街を共有」では
> なく、**各自が自分のプーブタウンを持ち、友達の街に “あそびに行く”** モデルにする
> （あつ森と同じ考え方）。これで「発展度が人で違う問題」は そもそも起きない。

### 何が「各自のもの」か（パーソナル）
- **自分のプーブタウン**：発展度・空き物件の再生・配置・景観
- **自分の家/部屋・畑・お店・もちもの・図鑑・称号・スキル・コンディション**
- → その子の遊び方で 世界が育つ（他人の進み具合に左右されない）

### 何が「みんな共通」か
- **時間（標準時計）**：朝→昼→夕→夜は **世界共通で同期**（一緒に遊ぶため）。
  時間は発展度と無関係なので、共通でも矛盾しない

### オンライン＝「訪問」でつながる
- 友達の街に **あそびに行く**／自分の街に **招く**。訪問中は **その街のオーナーの
  発展度・部屋・お店** が見える（＝その子の世界にお邪魔する）
- できること：
  - 🏠 **部屋/街を見せ合う**（発展度や飾りを見て「いいな！」＝刺激・あこがれ）
  - 🛒 **お店で売り買い**（自分の店で友達に売る／友達の店で買う）
  - 🤝 **共同クエスト**：誰かの街（ホスト）に集まって 一緒に達成、or 待ち合わせ広場
- **競争ではなく 見せ合い・助け合い**（ねらい§0）。発展度ランキングのような煽りはしない

### 任意：みんなの広場（ロビー）
- 待ち合わせ用の **共有スペース**を1つ置く案（オーナー無しの中立エリア）。
  ここで集合 → 誰かの街へ一緒に移動。必須ではない（まずは相互訪問だけでも成立）

### 実装メモ
- セーブは **各自の街データ**が基本（端末/サーバー）。訪問時は **ホストの街データを読む**
- 発展度・空き物件の状態は **個人スコープ**（共有・合算はしない）

## 17-3. セーブ機能（重要）

- **自動セーブ**：ねる/お店利用/クエスト達成などの区切りで自動保存
- **手動セーブ**：おうちでいつでも保存

- セーブ内容は §18 のデータモデル参照（プーブ・ニーズ・時間・もちもの・クエスト・称号・部屋・図鑑・お店すべて）
