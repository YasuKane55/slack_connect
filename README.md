# 🐤 ぷーぶ君の まちたんけん（プーブタウン）

> こどもプーブリカ オリジナルキャラ「ぷーぶ君」が主役の、独自通貨 **プーブ** で遊ぶ
> 子ども向け生活アドベンチャー。**街と関わる楽しさ**と、自分の**“好き”を見つける**
> きっかけになることをねらいにした、ブラウザ3D（Three.js）のゲーム企画です。
>
> 舞台「プーブタウン」は、**大阪・北加賀屋のリアルなお店をオマージュ**した街。
> 探検・お買い物・困りごと解決（クエスト）・農業・ものづくり・お店・スキル・称号、
> そして将来は **友達の街に遊びに行ける**オンラインへ。

最終更新: 2026-06-11 / 状態: **企画・設計＋ルック&フィール・デモ** 段階
