# ✨ シークレットガーデン ＝ エキスペリファーム（知的探究の農園）

> シークレットガーデン（連鎖クエストC5の到達点）の **正体**。
> いくつかの称号を獲得すると アンロックされる、**「実験・観察・考える」ための
> 自分だけの農園**。空き地から、掘って・建てて・植えて、特別な空間を育てる。
> ねらい§0（好きを見つける・自分で考える）の **集大成**。

最終更新: 2026-06-11 / v1

## 🔓 アンロック条件
- **称号を いくつか獲得**すると 解放（仮：**5個以上**／できれば 複数ジャンル）
  - ＝「いろんな“好き”を試した子」へのごほうび。特定1分野だけでなく 幅広く
- 解放の流れ：C5「ひみつの花園」を進める → げんさん/はる/わたるのヒント →
  かくし通路 → **エキスペリファーム 出現（常設）**
- 解放まで **正体は伏せる**（「なんだろう？」の余韻）。開けて初めて 正体が分かる

## 🌱 正体：エキスペリファーム とは
- **新しいことを 実験したり 考えたりする、知的探究のための農園**
- 「正解」も「ゴール」もない。**自分のペースで ためす・観察する・考える** 場所
- ぷーぶ君だけの **スペシャルな空間**を、ゼロから 育てていく

## 🏗 できること：自分だけの空間づくり（サンドボックス）
空き地状態から、自由に手を入れて 育てる（部屋づくりの“屋外・もっと自由”版）。
- ⛏ **地形をいじる**：穴を掘る／池をつくる／丘・小道をつくる
- 🪑 **設置する**：ベンチ／小屋／看板／アート／実験装置／かかし など
- 🌳 **植える**：木・花・作物・めずらしい植物（育てて 観察）
- 🎨 **かざる・レイアウト**：自由配置。自分の“好き”を詰め込む
- 💾 すべて **自分の街データに保存**（個人スコープ／§17-2c）

## 🔬 探究・実験のアクティビティ（ここならではの遊び）
- 🌾 **かけあわせ実験**：作物/花を かけあわせて **新種・変わった実**を発見
  → 植物図鑑の **特別枠**（ここでしか採れないもの）
- 🔎 **ふしぎ観察**：育て方（水/日かげ/肥料）を変えて **違いを観察・記録**（観察ノート）
- 💭 **問いを立てる**：ひなこ（こどもプーブリカ・哲学対話）と連動。
  「どうして？」のお題が出て、自分なりに **考えて 試す**
- 🛠 **発明・くふう**：水やり装置・しかけ などを 作って 試す（ものづくりの応用）
- 📓 **たんけんノート**：気づき・発見を記録（自己発見§1-3 とつながる）

## 🎯 ねらいとの関係（なぜ集大成か）
- §0「**好きを見つける**」を巡ってきた子が、最後に **自分だけの探究の場**を持つ
- 「**正解より 問い**」「**ためす・考える 楽しさ**」を 体現する場所
- ゴールがない＝**ずっと遊べる**自己表現/探究の庭（長く遊べる§17-2）

## 🧩 実装メモ
- ベースは **部屋づくり（room）の屋外・拡張版**：グリッドに 地形/設置/植物 を配置
- データ：`experiFarm = { terrain[], objects[], plants[], notes[] }`（個人セーブ）
- かけあわせ：簡単な 組み合わせ表（A×B→C）から。レアは低確率で新種
- まずは **置く・植える・掘る** の基本だけで成立 → 実験要素は後から拡張
- フェーズ：本体完成後の **やり込み/エンドコンテンツ**として実装でOK
